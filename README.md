# 🧩 Customer Segmentation using Data Mining & Machine Learning

This project applies clustering (K-Means) on customer purchase data to find meaningful segments.  
Used in marketing analytics and customer behavior modeling.

## 📊 Tech Stack
- Python 🐍  
- Pandas, NumPy for data processing  
- Scikit-learn for clustering  
- Matplotlib & Seaborn for visualization  

## 🚀 How to Run
1. Install dependencies:
```bash
   pip install -r requirements.txt
   
Put your data in data/customers.csv
Run clustering:
python src/segment_customers.py
   
Explore results in notebooks/analysis.ipynb
📈 Results
You’ll get clear visual clusters of customers based on spending patterns, frequency, and recency.
